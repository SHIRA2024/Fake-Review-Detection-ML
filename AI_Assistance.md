# AI AssistanceThis project was developed with the assistance of ChatGPT during different stages of the development process.ChatGPT was used to assist with:- Brainstorming and selecting the project idea.- Exploring and understanding the dataset.- Planning the exploratory data analysis (EDA).- Comparing different machine learning models.- Improving feature engineering and model optimization.- Building the Streamlit application.- Improving the project documentation, including the README and the final report.All code, experiments, model evaluation, and final decisions were implemented, executed, verified, and reviewed as part of the project development process.
